# Project Guidelines: React Frontend for Backend Integration

Dự án này là một ứng dụng React riêng lẻ được thiết kế để kết nối với Backend có sẵn.

## Project Overview
- **Tech Stack:** React, TypeScript, Vite, Tailwind CSS.
- **State Management:** React Context hoặc Hooks đơn giản.
- **Routing:** React Router DOM.
- **UI Components:** Lucide React (icons), Recharts (charts).

## Core Features
1. **Authentication:** Trang Login nhận User/Pass, lưu trữ Bearer Token.
2. **Main Layout:** Sidebar với 3 tùy chọn chính: Home (Dashboard), Users, Customers.
3. **Dashboard:** Hiển thị dữ liệu mẫu (smoke data) với biểu đồ hiện đại.
4. **Data Lists:** Trang danh sách Users và Khách hàng với dữ liệu giả định.
5. **Component Showcase:** Một trang riêng hiển thị các components dùng chung (Input, Dropdown, Multi-select, Date, Time, DataTables...).

## Coding Standards
- Sử dụng Functional Components và Hooks.
- Tailwind CSS cho styling.
- Tiếng Anh cho giao diện Website, Tiếng Việt cho tài liệu/comment nếu cần.
